# CropQuest
CropQuest is a web-based agricultural simulator that blends sustainability, economics, and strategy to teach real-world farm decision-making.

Players take on the role of a farmer and manage the complete farming cycle — from sowing crops to harvesting and selling — while handling soil health, climate risks, and financial challenges.

🎯 Problem Statement

Agricultural education is often theoretical and lacks practical exposure to real-world decision-making. Many learners do not fully understand:

Soil nutrient management

Crop rotation importance

Climate risks in farming

Financial challenges like loans and market fluctuations

Sustainable farming practices

CropQuest addresses this gap through an interactive simulation environment.

💡 Features

🌱 Complete farming cycle (Sowing → Harvest → Selling)

💰 Loan & bank reward system

🌍 Regional level progression

📊 Soil and crop information panel

🧑‍🏫 In-game guide and decision prompts

🏆 Profile & ranking system

🐄 Farm ecosystem elements (animals & infrastructure upgrades)
 

🧠 Learning Outcomes

Sustainable farming awareness

Financial risk management

Climate adaptability

Strategic thinking

Resource optimization

🛠 Tech Stack

Frontend: HTML, CSS, JavaScript

Backend : Node.js

Database: Firebase / Local Storage
